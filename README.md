## 📖상세 내용

Kubernetes와 Azure의 리소스를 이용하여 3Tier 웹 서비스를 구축한다.

**웹 서비스**

- DockerFile & Build
    - WAS Container
    - Web Container

**AKS**

- Service
    - web.yaml
    - was.yaml
- HPA
    - web-hpa.yaml
    - was-hpa.yaml
